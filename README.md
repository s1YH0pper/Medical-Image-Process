## 使用方法

1. 安装环境

   - 通过 pip 包管理导入环境

   ```bash
   pip install -r requirements_pip.txt
   ```

   - 通过 anaconda 或 miniconda

   ```bash
   conda create -n ENVNAME --file requirements_conda.txt
   ```

2. 运行对应目录下的`main.py`文件
